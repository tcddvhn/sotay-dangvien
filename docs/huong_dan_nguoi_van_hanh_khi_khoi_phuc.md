# Huong dan nguoi van hanh khi khoi phuc

Ngay cap nhat: 2026-04-15

## Khi nao dung lenh khoi phuc

Dung khi co mot trong cac dau hieu sau:

- mat node trong cay noi dung
- sai phan quyen
- admin khong luu duoc
- thu tu cac phan lon bi sai
- giao dien quan tri bi vo sau nang cap

## Cau noi de dung voi Codex

Dung dung cau:

```text
Khoi phuc lai he thong truoc day
```

Neu co nhieu bo backup, noi them:

```text
Khoi phuc lai he thong truoc day tu bo backup C:\backup-sotay\backup-truoc-phan-quyen-20260415-0930
```

## Viec anh/chị can chuan bi

1. Dam bao bo backup da ton tai day du.
2. Dam bao trong bo backup co file:
   - `manifest/backup_manifest.json`
   - `firestore/sotay_dulieu.json`
   - `source-code/`
3. Tam dung viec sua noi dung trong luc khoi phuc.
4. Neu can khoi phuc Apps Script/Google Sheet, phai co quyen truy cap tai khoan Google lien quan.

## Viec Codex se lam

- doc manifest
- doi chieu bo backup
- khoi phuc code
- khoi phuc Firestore
- bao cao ket qua

## Viec co the van can thao tac tay

- dang nhap vao Firebase Console
- import lai user auth neu pham vi rollback co lien quan auth
- doi chieu lai Apps Script/Google Sheet

## Sau khi khoi phuc xong

Anh/chị nen tu kiem tra:

1. Trang chu
2. Quan tri noi dung
3. Danh ba
4. 1 muc noi dung bat ky
5. chatbot/thong bao neu co lien quan
