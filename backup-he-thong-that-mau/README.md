# Bo backup he thong that mau

Thu muc nay la mau cau truc backup de anh/chị tao ban backup that o ngoai repo.

Khong nen dung truc tiep thu muc nay lam noi luu backup that, vi repo co the bi thay doi tiep.

Khi tao backup that, copy cau truc nay ra vi tri rieng, vi du:

```text
C:\backup-sotay\backup-truoc-phan-quyen-20260415-0930
```

Sau do dien day du cac file backup vao dung thu muc con.

Khi can khoi phuc, Codex se dua vao:

- `manifest/backup_manifest.json`
- `firestore/*.json`
- `source-code/`

de thuc hien quy trinh rollback.
