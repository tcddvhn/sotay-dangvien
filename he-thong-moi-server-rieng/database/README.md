# Database cho hệ thống mới

Thư mục này dành cho:

- thiết kế schema `SQL Server`
- script tạo bảng
- script migrate dữ liệu từ Firestore
- script seed dữ liệu ban đầu
- tài liệu mapping dữ liệu cũ sang dữ liệu mới

Các nhóm dữ liệu chính cần thiết kế:

- nội dung sổ tay
- danh bạ
- tài khoản quản trị
- vai trò và phân quyền
- khảo sát
- thống kê
- lịch sử chatbot
- đăng ký nhận thông báo
- cấu hình hệ thống
